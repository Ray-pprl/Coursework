Емулятор роботи дорожнього світлофора

Курсова робота з дисципліни "Інженерія програмного забезпечення" | 2 курс
Автор: Андрєєв Максим Денисович 


Опис проекту
Цей проєкт є консольним застосунком на мові C++, який моделює роботу дорожнього світлофора згідно з заданою умовою. Програма зчитує вхідні дані з файлу, імітує циклічну зміну сигналів (Червоний - Жовтий - Зелений  Жовтий) та підраховує кількість спрацювань кольорів до моменту, коли зелений сигнал загоряється в заданий раз.


Основний функціонал
* Моделювання циклу світлофора: точне повторення послідовності сигналів згідно з умовою задачі.
* Два режими роботи: візуальний (з кольоровою індикацією та затримками) та швидкий (миттєвий підрахунок без анімації).
* Кольорова індикація: автоматичне фарбування тексту в консолі (Червоний, Жовтий, Зелений) через Windows API.
* Файловий обмін: зчитування вхідного значення з input.txt, запис результатів у output.txt.
* Валідація даних: перевірка коректності вхідного числа (1 ≤ i ≤ 100) з інформативним повідомленням про помилку.
* Підрахунок статистики: автоматичний підрахунок кількості спрацювань червоного та жовтого кольорів.

Технологічний стек
* Мова програмування: C++
* Середовище розробки: Visual Studio
* Платформа: Windows
* Стандартна бібліотека: <iostream>, <fstream>
Як запустити:
 1)Створіть консольний проєкт у Visual Studio.
 2)Додайте файл main.cpp з кодом програми.
 3)У папці проєкту створіть текстовий файл input.txt і запишіть у нього ціле число від 1 до 100.
 4)Зберіть рішення (Build → Build Solution) та запустіть (Ctrl+F5).
 5)У консолі оберіть режим роботи: 1 — візуалізація, 2 — швидкий результат.
 Приклад роботи:
 Вхідні дані (input.txt):
 *3*
 Вихідні дані (output.txt):
 *3 5*
 Тестування:
 Проведено тестування трьома методами: Black Box, White Box, Unit. Перевірено коректність роботи на граничних значеннях (1, 100), типових значеннях та некоректних  даних (0, 101, -5). Усі тести пройдено успішно.
