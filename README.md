Емулятор роботи дорожнього світлофора

Курсова робота з дисципліни "Розробка програмного забезпечення" | 2 курс
Автор: Андрєєв Максим Денисович 


Опис проекту
Цей проект є консольним додатком на мові C++, який симулює циклічну роботу класичного дорожнього світлофора. Програма наочно демонструє зміну фаз руху (Червоний -> Жовтий -> Зелений -> Жовтий) з використанням псевдографіки та колірної індикації в консолі Windows.

Основний функціонал
* Реалістичний цикл: Точне повторення алгоритму роботи реального світлофора.
* Візуалізація: Відмальовування графічного силуету світлофора в консолі.
* Колірна індикація: Автоматичне фарбування активного сигналу (Червоний, Жовтий, Зелений) через WinAPI.
* Таймер відліку: Відображення реального часу (в секундах) до наступної зміни сигналу.

Технологічний стек
* Мова програмування: C++
* Середовище розробки: Visual Studio
* Платформа: Windows
